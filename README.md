# Seungjun's Personal Agent Suite

A collection of AI-powered personal productivity tools built and deployed during a 5-day sabbatical — with zero prior coding experience.

**Live:** [https://fpfe.github.io/my-agent/](https://fpfe.github.io/my-agent/)

---

## Apps

| App | Description |
|-----|-------------|
| **Sabbatical Command Center** | Central dashboard and home hub connecting all tools, with a job search pipeline board and daily quick-access panel. |
| **Daily Planner** | Sunsama-inspired daily planner for structuring your day, tracking tasks, and journaling mood and progress. |
| **Todo List** | Lightweight task manager with priority levels, due dates, and category filters. |
| **Job Search Agent** | AI-assisted job search tracker for managing applications, researching companies, and drafting cover letters. |
| **AI Learning Tracker** | Personal knowledge map for tracking progress in AI/ML topics, with built-in self-quiz functionality. |
| **Prompt Bank** | Searchable library for saving, organizing, and reusing your best prompts across Claude, ChatGPT, and Cursor. |

## Getting Started

Visit the live site: [https://fpfe.github.io/my-agent/](https://fpfe.github.io/my-agent/)

To run locally, clone the repo and open `index.html` in your browser — no build step required.

```bash
git clone https://github.com/fpfe/my-agent.git
cd my-agent
open index.html
```
